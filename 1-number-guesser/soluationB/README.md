# Number Guesser Game Project


## Description
This project is a game where users can guess a number between 1 and 100. The user will be given a hint if a number is too low or too high. The user will also will be able to see the number they have guessed. The user will be able to reset the game at any time using key "q"


## How to Run
1. Clone down this repo, and cd into the project
2. Run `pip install -r requirements.txt` in your comand line
3. Run `python number_guesser.py` in your command line
