# Rock Paper Scissors Game

this project is a python-based implementation of the popular game "Rock, Paper, Scissors". It provides an opprtunity to play aginst a computer , which ,akes it moves based on random choice. The game continues indefinitly untill the user stops. then it shows a stat

## Requirements
1. Python 3.6+: This Project uses several Python 3.6+ features like fstrings and type hinting

To install the reqirements:

```bash
pip install -r requirememts.txt
```

## Running the Project
To run the project

```python
python src/game.py
```

The game can be played by running the `game.py` script. The game continues untill the user decides to quit by entering "q".
